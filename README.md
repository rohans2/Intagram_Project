# Intagram_Project
An Instagram- Like App which covers majority of features offered by instagram including-
->Profile Picture
->Posts
->Likes 
->Comments 
->Saving Pictures 
->Recent Notifications recieved when someone likes or comments on a post
->Search usernames as well as hashtags.
